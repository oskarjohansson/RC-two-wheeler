# RC two wheeler
A 4 kW RC car with overkill BLDC motors and two VESCs and a 5 kW LiPo.  

## Hardware
- Flipsky VESC v4.12 x 2
- KEDA 63-64 190KV 2kW x 2
- Arduino Nano x2
- nRF24 x2
- 3S 60C 4000 mAh Turnigy Heavy duty LiPo
- Joystick
